## SYSTEMKONTEXT

Das System hat 3 UI Ebenen:

### 1. User Browser UI (27” PC)
- Voting System
- Ergebnis Eingabe
- Aufgaben / Soulmask Management
- Status Übersicht

### 2. TV UI (50–55” Display)
- Event Dashboard
- Leaderboard
- Voting View (6–8 Spiele)
- Fullscreen Glücksrad Animation
- Live Status

### 3. Admin UI (Mini Control Panel)
- Start/Stop Voting
- Mode Switching
- Turnier Kontrolle
- System Overrides

---

## WICHTIGE DESIGN ANFORDERUNGEN

- UI ist Kernprodukt, nicht Nebenfeature
- TV ist groß, sichtbar aus 2–5m Entfernung
- Browser UI ist schnell und klickoptimiert
- Admin UI ist minimal und funktional
- alles muss extrem klar und einfach sein
